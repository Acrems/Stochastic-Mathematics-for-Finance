#  Stochastic Mathematics for Finance

An 8-week project exploring the mathematical foundations of quantitative finance through theory, programming assignments, and numerical simulations.

The repository contains weekly implementations of concepts such as probability theory, stochastic processes, Monte Carlo methods, Brownian motion, stochastic differential equations, and option pricing using Python (with a later C++ implementation for performance benchmarking).

---

## Topics Covered

- Advanced Probability
  - Conditional Probability
  - Bayes' Theorem
  - Expected Value
- Markov Chains
- Random Walks
- Monte Carlo Simulation
- Variance Reduction Techniques
- Central Limit Theorem
- Brownian Motion

Each week's folder contains:
- Python implementations
- Experiment scripts/notebooks
- Supporting notes and derivations (where applicable)

---

## Key Implementations

- Monte Carlo integration with Antithetic Variates
- Markov Chain simulator
- Random Walk simulator
- Brownian Motion path generator
- Euler-Maruyama SDE solver

---

## Learning Objectives

This project focuses on building an intuitive and computational understanding of stochastic processes commonly used in quantitative finance by:

- Implementing mathematical concepts from scratch
- Validating theoretical results through simulation
- Developing reusable numerical methods
- Applying probability and statistics to financial modeling problems
